Projectile Motion Simulation

This project demonstrates the simulation of projectile motion using Python.
It is designed as a teaching and learning tool for secondary school and undergraduate physics.

Features
- Visualization of projectile trajectories
- Effect of launch angle and initial velocity
- Physics-based equations of motion

Tools Used
- Python
- NumPy
- Matplotlib
  
 Purpose
This project supports physics education by helping learners visualize abstract kinematics concepts
through interactive computational simulations.
